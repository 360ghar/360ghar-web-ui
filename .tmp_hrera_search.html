Password Policy

1.Password must at least be 8 characters.
2.Password must not be the same as the previous 3 passwords.
3.Password must contain Uppercase, lower case, number and Special Characters.
4.If you do not set the password correctly, a message will be flashed "Your password is not as per Password Policy." This implies that the password needs to be set again as per the guidelines stated above.
